# TERMINAL_SNAKE

A sharp, minimalist Snake game built with vanilla HTML5 Canvas, CSS3, and JavaScript. Self-contained and dependency-free.

## Features
- **Biome System**: Visual theme shifts every 50 points across 5 distinct biomes.
- **Ghost Trail**: Advanced motion-blur effect with zero collision impact.
- **Combo Multiplier**: Score bonuses for rapid food consumption (up to ×5).
- **Portal Walls**: WARP MODE unlocks at 150 points for edge teleportation.
- **Synthesized Audio**: Pure Web Audio API sound engine with zero external assets.

## Instructions
1. Open `index.html` in any modern web browser.
2. Survive and grow to unlock WARP MODE protocol.
