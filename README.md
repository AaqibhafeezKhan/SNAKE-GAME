# TERMINAL_SNAKE Protocol

A sharp, minimalist Snake game built with vanilla HTML5 Canvas, CSS, and JavaScript. Self-contained in a single file for easy deployment.

## Features
- **Biome System**: Visual themes shift every 50 points (VOID, TECH, ACID, GOLD, MONO).
- **Ghost Trail**: Motion-blur effect tracking the snake's path.
- **Combo Multiplier**: Earn up to 5x score by eating food quickly.
- **Obstacle Pulse**: Warning system before obstacles relocate.
- **Mobile Support**: Includes touch-friendly D-pad controls.

## How to Play
1. Open `index.html` in any browser.
2. **Desktop**: Use Arrow Keys or WASD to move.
3. **Mobile**: Use the on-screen D-pad buttons.
4. Eat food to grow and increase your score. Avoid walls, obstacles, and yourself!
